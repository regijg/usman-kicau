# 🐦 USMAN - Usaha Manuk

Website untuk usaha penjualan burung kicau dan pakan burung berkualitas.

## Fitur Utama

- 🐦 **Katalog Burung**: Menampilkan berbagai jenis burung yang tersedia
- 🎰 **Pakan Burung**: Informasi tentang pakan berkualitas (Ulat Hongkong, Jangkrik)
- 📞 **Kontak Langsung**: Integrasi WhatsApp dan telepon untuk komunikasi mudah
- 📱 **Responsive Design**: Tampilan sempurna di semua perangkat
- 🎨 **Modern UI**: Desain modern dengan Tailwind CSS

## Teknologi

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS 3
- **Language**: TypeScript
- **UI Components**: React

## Instalasi

1. Clone atau download project ini
2. Install dependencies:

```bash
npm install
```

3. Jalankan development server:

```bash
npm run dev
```

4. Buka [http://localhost:3000](http://localhost:3000) di browser Anda

## Build untuk Production

```bash
npm run build
npm run start
```
